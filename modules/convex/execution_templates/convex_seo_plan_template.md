## Phase 4: Technical SEO Checklist (Week 4)

### 4.1 On-Page Technical Optimization

```
[ON-PAGE REQUIREMENTS - Every page must have]

Meta Tags:
- [ ] Title tag 50-60 characters, includes primary keyword
- [ ] Meta description 150-160 characters, compelling CTR (2-3% improvement target)
- [ ] H1 tag (only 1 per page), includes target keyword
- [ ] H2-H6 hierarchy (proper structure)

Internal Linking:
- [ ] 3-5 internal links per page (to related content)
- [ ] Link anchor text descriptive (not "click here")
- [ ] Links to pillar pages from subtopics
- [ ] Links to related clusters

Image Optimization:
- [ ] Image alt text includes relevant keywords
- [ ] File name optimized (not "image123.jpg")
- [ ] Compressed for web (<200KB per image)
- [ ] Descriptive captions (improves user experience)

Schema Markup:
- [ ] FAQ schema (for FAQ pages)
- [ ] Article schema (for blog posts)
- [ ] BreadcrumbList schema (for navigation)
- [ ] Organization schema (homepage)

Core Web Vitals:
- [ ] Largest Contentful Paint (LCP) <2.5 seconds
- [ ] First Input Delay (FID) <100ms
- [ ] Cumulative Layout Shift (CLS) <0.1
- [ ] Page load time <3 seconds (90th percentile)

Mobile Optimization:
- [ ] Responsive design (tests on mobile)
- [ ] Touch-friendly buttons (48px minimum)
- [ ] Readable font size (16px minimum body text)
- [ ] No mobile-only popups blocking content
```

### 7.2 Page Optimization Process

```
[UNDERPERFORMING PAGE FIXES]

If page ranks 11-20 for target keyword:
1. Review current content vs. top 3 competitors
2. Identify missing elements (formatting, depth, proof, examples)
3. Rewrite/expand with missing elements
4. Add internal links from pillar page
5. Reach out to relevant websites to request links
6. Re-test in 30 days

If page ranks 21-50:
1. Consider target keyword feasibility (search volume enough?)
2. If too competitive, shift to long-tail variation
3. Expand content depth by 50%
4. Add missing sections competitors have
5. Improve technical optimization

If page not ranking (below 50):
1. Verify article is actually indexed (site:yourdomain.com)
2. Check for noindex tag (remove if present)
3. Add strategic internal links from high-authority pages
4. Submit XML sitemap update
5. Wait 60 days and re-evaluate
6. If still no ranking, consider pruning or merging
```
