## ğŸ”Œ API Endpoints Structure

### Email System
```
POST   /api/email/webhook              - Gmail webhook receiver
POST   /api/email/parse                - Parse email content
POST   /api/email/auto-reply           - Generate & send auto-reply
GET    /api/clients/[id]/emails        - Get client emails
POST   /api/clients/[id]/emails/link   - Link email address
```

### Client Management
```
POST   /api/clients                    - Create new client
GET    /api/clients/[id]               - Get client details
PUT    /api/clients/[id]               - Update client
GET    /api/organization/clients       - Get all org clients
DELETE /api/clients/[id]               - Delete client
```

### Assets
```
POST   /api/clients/[id]/assets/upload - Upload asset
GET    /api/clients/[id]/assets        - Get all assets
DELETE /api/clients/[id]/assets/[aid]  - Delete asset
PUT    /api/clients/[id]/assets/[aid]  - Update asset metadata
```

### Personas
```
GET    /api/clients/[id]/persona       - Get current persona
POST   /api/generate/persona           - Generate/update persona
GET    /api/clients/[id]/persona/history - Persona versions
POST   /api/clients/[id]/persona/export - Export persona
```

### Mind Maps
```
GET    /api/clients/[id]/mindmap       - Get mind map
POST   /api/clients/[id]/mindmap/update - Update mind map
POST   /api/clients/[id]/mindmap/export - Export mind map
GET    /api/clients/[id]/mindmap/history - Version history
```

### Marketing Strategies
```
GET    /api/clients/[id]/strategy      - Get strategy
POST   /api/generate/strategy          - Generate/update strategy
GET    /api/clients/[id]/strategy/platforms - Platform strategies
POST   /api/clients/[id]/strategy/export - Export strategy
```

### Campaigns
```
GET    /api/clients/[id]/campaigns     - Get all campaigns
POST   /api/clients/[id]/campaigns     - Create campaign
PUT    /api/clients/[id]/campaigns/[cid] - Update campaign
DELETE /api/clients/[id]/campaigns/[cid] - Delete campaign
GET    /api/clients/[id]/campaigns/[cid]/calendar - Calendar view
POST   /api/clients/[id]/campaigns/duplicate - Duplicate campaign
```

### Hooks & Scripts
```
GET    /api/clients/[id]/hooks         - Get all hooks
POST   /api/generate/hooks             - Generate hooks
GET    /api/hooks/search               - Search hooks
POST   /api/hooks/favorite             - Save favorite
DELETE /api/hooks/[hid]                - Delete hook
```

### Images
```
GET    /api/clients/[id]/images        - Get all images
POST   /api/generate/images            - Generate DALL-E concepts
PUT    /api/clients/[id]/images/[iid]  - Update image
DELETE /api/clients/[id]/images/[iid]  - Delete image
POST   /api/clients/[id]/images/regenerate - Regenerate concepts
```

### Billing & Subscription
```
GET    /api/subscription/[orgId]       - Get subscription
POST   /api/subscription/upgrade       - Upgrade plan
POST   /api/subscription/downgrade     - Downgrade plan
POST   /api/billing/webhook            - Stripe webhook
GET    /api/billing/invoices           - Get invoices
POST   /api/billing/cancel             - Cancel subscription
```
