## Error Taxonomy (used by `reproduce-bug` node)

Categorise the bug before writing a failing test:

| Category         | Pattern                                             |
| ---------------- | --------------------------------------------------- |
| `TYPE_ERROR`     | TypeScript type mismatch, undefined property access |
| `NULL_REFERENCE` | Cannot read properties of null/undefined            |
| `ASYNC_RACE`     | Promise not awaited, race condition                 |
| `IMPORT_ERROR`   | Module not found, circular dependency               |
| `AUTH_ERROR`     | JWT invalid, token expired, CORS blocked            |
| `DB_ERROR`       | SQLAlchemy constraint violation, connection error   |
| `NETWORK_ERROR`  | Fetch failed, timeout, DNS resolution               |
