## Overview

Generates personalized marketing content using Claude Opus 4.5 with Extended Thinking.

## Responsibilities

1. **Content Generation**
   - Personalized emails for warm leads
   - Follow-up sequences
   - Campaign content

2. **Extended Thinking**
   - Complex reasoning (5000-10000 token budget)
   - Strategic content planning
   - Deep personalization

3. **A/B Testing Support**
   - Multiple content variants
   - Subject line optimization
   - CTA testing

## AI Configuration

- Model: `claude-opus-4-5-20251101`
- Extended Thinking: Enabled (5000-10000 tokens)
- Cost: ~$0.10-0.20 per content generation

## CLI Commands

```bash
npm run content-agent            # Generate content
npm run generate-content         # Content generation
```
