## Skill Development Notes

### Version 1 Scope
Every skill clearly states what's included in V1 vs. what's Post-V1.

### HUMAN_GOVERNED
Skills marked HUMAN_GOVERNED are advisory only - they don't make decisions.
The founder/user always decides.

### Extended Thinking
Skills that use Extended Thinking (deep analysis) note this in their capabilities.
Budget: Usually 5,000-10,000 tokens for complex analysis.

### Error Handling
Each skill includes "Error Handling" section explaining failure modes.
