## Skill Development Guidelines

### Creating a New Skill

1. **Create Markdown File**: `.claude/commands/skill-name.md`

2. **Document Structure**:
   ```markdown
   # Skill Name

   ## Purpose
   Brief description of what this skill does

   ## Usage
   /skill-name [param1] [param2]

   ## Parameters
   - param1: Description (required/optional)
   - param2: Description

   ## Implementation
   Code or pseudocode showing how it works

   ## Example
   /skill-name example-value another-value

   ## Output
   What the skill returns or produces
   ```

3. **Register in Claude Code** (if managed skill):
   - Add to `.claude/skills/` if using managed skills plugin
   - Or keep in `.claude/commands/` for local-only

4. **Test the Skill**:
   ```bash
   # In Claude Code CLI
   /skill-name test-params
   ```
