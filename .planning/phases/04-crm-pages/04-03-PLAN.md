## Token Mapping Reference

```
Semantic (always migrate):
red-* → error-*
green-* → success-*
blue-* → info-*
yellow-* → warning-*
orange-* → accent-*
amber-* → warning-* (context-dependent)

Structural (migrate to tokens):
gray-50/100/200 → bg-bg-subtle (light theme) or keep
gray-300/400 → text-text-muted
gray-500/600 → text-text-secondary
gray-700/800/900 → text-text-primary or bg-bg-*
slate-* → bg-bg-base/raised/card

Decorative (keep as-is):
purple-*, pink-*, cyan-*, indigo-* in charts/gradients
```

## Acceptance Criteria

- [ ] Status indicators use design tokens
