## Token Mappings

Standard mappings from previous phases:

### Text Colors
- `text-gray-900` → `text-text-primary`
- `text-gray-700` → `text-text-secondary`
- `text-gray-600/500/400` → `text-text-muted`
- `text-slate-*` → equivalent text-text-* token

### Backgrounds
- `bg-gray-50` → `bg-bg-hover`
- `bg-gray-100` → `bg-bg-hover`
- `bg-gray-900` → `bg-bg-base`
- `bg-slate-*` → equivalent bg-bg-* token

### Borders
- `border-gray-200` → `border-border`
- `border-gray-300` → `border-border-subtle`

### Semantic Colors
- `red-*` → `error-*`
- `green-*` → `success-*`
- `yellow-*` → `warning-*`
- `blue-*` → `info-*` or `accent-*` (for links/CTAs)
