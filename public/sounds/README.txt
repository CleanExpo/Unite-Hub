Notification Sound Placeholder
==============================

This folder should contain the notification.mp3 file for the chat widget.

To add the notification sound:
1. Add a file named "notification.mp3" to this folder
2. Use a short, pleasant notification sound (< 1 second)
3. Keep the file size small (< 50KB) for quick loading

Free notification sounds available at:
- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/

The chat widget will play this sound when new messages arrive.
