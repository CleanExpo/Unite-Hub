## Production Notes

### Recording Requirements

**Screen Recording:**
- Resolution: 1920x1080 minimum
- Frame rate: 30fps
- Format: MP4 with H.264

**Audio:**
- Professional voiceover
- Clear narration, moderate pace
- Background music: subtle, non-distracting

**Branding:**
- Use Unite-Hub logo in intro/outro
- Consistent color scheme
- Professional transitions

### Editing Guidelines

1. Keep cuts tight - no dead air
2. Highlight mouse cursor for actions
3. Use zoom for important details
4. Add callouts for key features
5. Include captions for accessibility

### Hosting

- Upload to YouTube (unlisted or public)
- Embed in help center
- Link from onboarding emails
- Available in app via "?" help icon
