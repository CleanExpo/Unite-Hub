# Copywriting & Business Consistency Enhancement Plan

**Date**: 2025-12-03
**Scope**: System-wide copywriting methodology + Business consistency framework
**Impact**: All content generation, client profiles, staff workflows, website copy
**Status**: PLANNING PHASE (awaiting approval)

---

## Executive Summary

This plan integrates four interconnected systems based on your provided methodology:

1. **Voice of Customer (VOC) Research Engine** - Real customer quotes from forums/reviews
2. **Competitor Analysis System** - Website structure analysis of top players
3. **Conversion Copywriting Engine** - Customer-quote-infused, jargon-free copy
4. **Business Consistency Framework** - NAP consistency, schema, GEO optimization

These systems apply to:
- **Synthex/Unite-Hub marketing site** - Our own copy
- **Client websites** - Copy we generate for managed services clients
- **Staff workflows** - Research tools and generation interfaces

## Your Prompts Mapped to Implementation

### Prompt 1: VOC Research → VOCResearchAgent

| Your Requirement | Implementation |
|------------------|----------------|
| "Analyze audience using business data dump" | VOCResearchAgent.loadBusinessContext() |
| "Research real message boards, review sites, forums" | Playwright MCP + Exa MCP web scraping |
| "Find exact words they use" | Claude extraction with "quotes only" instruction |
| "Not summaries, not interpretation—real words" | extractQuotes() preserves verbatim text |
| "Organize into categories" | 5 categories: pain points, symptoms, dreams, failed solutions, buying decisions |
| "Highlight patterns (gold)" | identifyPatterns() flags quotes appearing 3+ times |

### Prompt 2: Competitor Analysis → CompetitorAnalyzer

| Your Requirement | Implementation |
|------------------|----------------|
| "Analyze top companies in niche nationwide" | analyzeCompetitor(url) for top 5-10 national players |
| "Go through websites page by page" | Playwright crawls Homepage, About, Services, Contact |
| "List sections they include" | extractPageStructure() returns section names + order |
| "Order those sections appear in" | Array with position indices |
| "Identify unique pages/sections" | identifyUniqueFeatures() compares against your site |
| "Structure report by page type" | JSONB output grouped by page |

### CustomGPT Instructions → ConversionCopywritingEngine

| Your Requirement | Implementation |
|------------------|----------------|
| "Write one website page at a time on request" | /api/copywriting/generate/page |
| "Proven conversion structures" | Template: Hero → Problem → Value Props → Proof → Process → Pricing → FAQ → CTA |
| "Clearly labeled sections" | JSONB output with section keys |
| "Work customer quotes naturally into copy" | integrateCustomerVoice() weaves quotes without literal quotes |
| "Ultra-clear, conversational tone" | Tone validation + jargon detector |
| "Avoid jargon, clichés, overused lines" | Blocklist of banned phrases + AI review |
| "Only verified details (no made-up claims)" | Cross-reference with business_consistency_master |
| "Output only final copy—no explanations" | Response format: copy only, no meta |

### Consistency Framework → BusinessConsistencyService

| Framework Element | Implementation |
|-------------------|----------------|
| Tier 1 Critical (NAP) | legal_business_name, street_address, primary_phone |
| Tier 2 Essential | website_url, email_address, business_hours, primary_category |
| Tier 3 Important | descriptions, service_areas, payment_methods, social_profiles |
| Tier 4 Australia-specific | abn, acn, license_numbers |
| Platform Priority List | citation_listings table with tier 1-5 platforms |
| Schema Markup | Auto-generated JSON-LD from master document |
| GEO Optimization | sameAs links, entity graph, AI citation readiness |
| Audit Checklist | /api/copywriting/consistency/audit returns score + issues |
